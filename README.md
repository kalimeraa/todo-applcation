# Todo Application

### API Architecture
- Runs on :5000
- [POST] => api/todos/create
- [GET] => api/todos
- Chai for unit test

### Frontend Architecture
- Runs on :80
- Vue | Vuex | Cypress for e2e test | Jest for unit test

## Stack
* **Docker**
* **Github Actions**
* **Vue.js**
* **Node.js**
* **MongoDB**
* **Bootstrap**


### Requirements for local installation

- Docker
- GIT

### Installation
```
  docker-compose up -d
```
